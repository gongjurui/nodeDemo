## 如何设置内网
- npm config set registry http://172.18.0.199
## 全局安装内网转化
- npm install nrm -g
- 添加珠峰源
```
npm add zhufeng http://172.18.0.199
```
- 使用源
```
nrm use zhufeng
```
- 显示所有源
```
nrm ls
```
